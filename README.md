## **Installation Guide**  
### Prerequisites  
Before installing, ensure you have:  
- **Java 17** or later installed  
- **Maven** installed
- **MariaDB** installed   
- **Browser** (e.g., Chrome, Firefox)
<h3 align="left">App Installation</h1>
1. Clone repository:

 ```sh
  git clone https://github.com/umuttig37/CarRentalWeb.git
  ```
